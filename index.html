使用方法：

1、引入js文件：
./js/postbird-mp4-to-blob.js
2、调用插件：
var url = './video/v0-new.mp4'; // 视频url
var mimeCodec = 'video/mp4; codecs="avc1.640028, mp4a.40.2"'; // 编码格式
PostbirdMp4ToBlob.init('#video',url,mimeCodec); // #video 是选择器
